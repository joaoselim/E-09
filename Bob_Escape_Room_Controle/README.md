# Bob Francis Clean 2 - The Great Escape

## Descrição

Este projeto final é um jogo 2D desenvolvido em Java. O jogo tem foco em exploração, interação com objetos e resolução de enigmas.

O jogador controla o Bob, um faxineiro de hotel que precisa encontrar a chave do prédio para conseguir sair. Durante o jogo, o personagem explora diferentes cenários, coleta itens e resolve enigmas para avançar na história.

## Objetivo do Jogo

O objetivo principal do jogo é explorar o hotel, encontrar pistas, resolver enigmas e obter a chave final para conseguir sair do prédio.

A progressão do jogo envolve:
- Explorar o saguão e corredores do hotel;
- Encontrar chaves e itens importantes;
- Interagir com objetos do cenário;
- Resolver enigmas;
- Abrir portas e acessar novos ambientes;
- Descobrir a senha do cofre;
- Coletar a chave final do hotel.

## Controles

| Tecla | Ação |
|------|------|
| A | Andar para a esquerda |
| D | Andar para a direita |
| Espaço | Pular |
| W | Interagir com objetos, portas e locais específicos |
| E | Abrir ou fechar a visualização de imagem/item na tela |
| S | Salvar o jogo |

## Uso de Inteligência Artificial no Projeto

A Inteligência Artificial foi utilizada como ferramenta de apoio durante o desenvolvimento do projeto, principalmente nas seguintes partes:

- Criação e projeção da interface gráfica;
- Geração de imagens de parte dos cenários;
- Recorte dos sprites do personagem e projeção das animações;
- Auxílio na utilização de arquivo;
- Apoio no tratamento de erros durante o desenvolvimento;
- Auxílio na correção de problemas relacionados à carregamento de imagens e funcionamento das interações.

As Inteligências Artificiais ultilizadas foram ChatGPT e Deepseek.
